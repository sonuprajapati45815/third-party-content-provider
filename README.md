## Environment Variables

Set the following environment variables before running the project:

- `REDIS_HOST` (e.g., redis)
- `REDIS_PORT` (e.g., 6379)
- `NEWS_API_KEY` (e.g., c1a233d12a44416c84076894d3ed02e1)
- `WEATHERAPI_KEY` (e.g., 0533299ca37745ffbce52129250206)
- `MODEL_NAME`
- `GEMINI_API_KEY`
- `OPENAI_API_KEY`
- `DEFAULT_LANGUAGE` (e.g., en)
